# Quick Start

This page gives a compact overview of the first actions to take with Honey Badger 5.0. Use it when you need a fast, reliable startup path.

## What to do first

Use this page as a short roadmap for the first time you start Honey Badger 5.0.

- Read [Safety](Safety.md) before you power on the robot.
- Review [Setup](Setup.md) for package contents, network setup, and charging.
- Follow [FirstStartup](FirstStartup.md) for the first startup checklist.
- Use [ControlConsole](ControlConsole.md) for controller and GUI operation details.

- [Safety](Safety.md) — required safety rules and emergency procedures.
- [Setup](Setup.md) — setup references and connection details.
