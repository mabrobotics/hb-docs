# Quick Start

This page gives a compact overview of the first actions to take with Honey Badger 5.0. Use it when you need a fast, reliable startup path.

## What to do first

Use this page as a short roadmap for the first time you start Honey Badger 5.0.

- Read `GettingStarted/Safety.md` before you power on the robot.
- Review `GettingStarted/Setup.md` for package contents, network setup, and charging.
- Follow `GettingStarted/FirstStartup.md` for the first startup checklist.
- Use `GettingStarted/ControlConsole.md` for controller and GUI operation details.

## Why these pages

- `GettingStarted/Safety.md` — required safety rules and emergency procedures.
- `GettingStarted/Setup.md` — setup references and connection details.
- `GettingStarted/FirstStartup.md` — step-by-step startup flow.
- `GettingStarted/ControlConsole.md` — Steam Deck control guidance.
